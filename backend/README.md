## Tables
Para atualizar as tabelas na base de dados, execute este comando.

```bash
    make create-database-tables
```

## Dependências
Para atualizar as dependências, execute este comando.

```bash
    make update-dependencies
```

## Testes
Para executar os testes, execute este comando.

```bash
    make test
```

Para abrir o relatório dos testes:
```bash
    make open-test-report
```

Para abrir o relatório do coverage:
```bash
    make open-test-coverage
```
